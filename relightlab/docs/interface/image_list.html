Image list

The image tab allows you to explore the photos.

You can have the list of the images and a single image shown or you can see the all the images at once using the icons on the top right

A checkbox for each photo is checkable and when not checked the image is not considered part of the dataset: if you load a file containing the light directions the application requires the number of checked images to be equal to the number of lights in the .lp file or dome.

An 'eye' icon on the right of the file name is used instead to remove some images from the computation of RTI's and normals.

You can also select the images clicking on the images (shift and control click allows you to select multiple images) and rotate them all at once using the 'rotate' icons.
